# Gzip

## files

### Compress file

Run compress file with Gzip.

```sh title:"Gzip Run Compress File"
gzip "$path"
```
<!-- cheat
var path
-->

### Decompress file

Run decompress file with Gzip.

```sh title:"Gzip Run Decompress File"
gzip -d "$gz_file"
```
<!-- cheat
var gz_file
-->

### Keep original

Run keep original with Gzip.

```sh title:"Gzip Run Keep Original"
gzip -k "$path"
```

var path
-->
