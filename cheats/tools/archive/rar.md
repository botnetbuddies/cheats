# RAR

## create

### Create rar

Create rar with RAR.

Create a RAR archive from a directory.

```sh title:"RAR Create Rar"
rar a "$archive_name.rar" "$directory"
```

## extract

### Extract rar

Extract rar with RAR.

Extract a RAR archive with full paths.

```sh title:"RAR Extract Rar"
unrar x "$archive_file"
```
