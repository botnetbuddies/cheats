# 7z

## create

### Password archive

Dump password archive with 7z.

Create a password-protected 7z archive.

```sh title:"7z Dump Password Archive"
7z a "$archive_name.7z" -p"$archive_pass" "$file"
```
<!-- cheat

### Password archive with encrypted names

Dump password archive with encrypted names with 7z.

Create a password-protected 7z archive and encrypt filenames.

```sh title:"7z Dump Password Archive with Encrypted Names"
7z a "$archive_name.7z" -p"$archive_pass" -mhe=on "$file"
```

## extract

### Extract archive

Extract archive with 7z.

Extract a 7z archive.

```sh title:"7z Extract Archive"
7z x "$archive_file"
```
