# X11 Enumeration

Exposed X11 can reveal display metadata, windows, screenshots, and sometimes input control when access control is disabled.

## Prerequisites

| Requirement | Detail |
|-------------|--------|
| TCP 6000 | Requires X11 reachability |
| Display number | Port usually maps to display number |
| X tools | Commands assume common X11 utilities are installed |

## Linux

### Access check

#sh #nmap #x11

Check for anonymous X11 access.

```sh title:"Check anonymous X11 access"
nmap -sV --script x11-access -p "$rport" "$rhost_ip"
```
<!-- cheat
var rhost_ip
var rport := 6000

### Display info

#sh #x11

Query X11 display information.

```sh title:"Query X11 display info"
xdpyinfo -display "$rhost_ip:$display"
```
<!-- cheat
var rhost_ip

-->

### Window tree

#sh #x11

List the X11 root window tree.

```sh title:"List X11 root window tree"
xwininfo -root -tree -display "$rhost_ip:$display"
```

### Keystroke capture

#sh #x11

Capture X11 keystrokes when the display allows access.

