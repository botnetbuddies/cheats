# LaZagne

LaZagne extracts stored credentials from supported local applications and user profile data.

## Linux

### All modules

#sh #lazagne #credentials

Run all LaZagne modules.

```sh title:"Run LaZagne all modules"
python3 laZagne.py all
```

